# TASK: Mobile Chat View

## Goal
Create mobile chat interface with message list, streaming responses, code blocks, input area, and connection status indicators.

## Scope

### Allowed
- packages/mobile/src/views/Chat/**
- packages/mobile/src/components/MessageBubble/**

### Forbidden
- packages/core/**
- packages/tui/**

## Requirements
1. Scrollable message list with auto-scroll on new messages
2. Streaming response rendering with typing indicators
3. Code blocks with syntax highlighting and copy button
4. Touch-optimized input area with send button
5. Connection-aware UI (offline/reconnecting indicators)
6. Message timestamps and read receipts

## Definition of Done
- [ ] Chat interface renders messages
- [ ] Streaming responses display in real-time
- [ ] Code blocks highlighted with copy
- [ ] Input sends messages correctly
- [ ] Connection status shown
