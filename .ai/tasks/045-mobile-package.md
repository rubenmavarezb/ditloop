# TASK: Mobile Package

## Goal
Create new packages/mobile with Vite, React DOM, Tailwind, PWA manifest, service worker, and connection setup for server communication.

## Scope

### Allowed
- packages/mobile/src/**
- packages/mobile/package.json
- packages/mobile/vite.config.ts
- packages/mobile/tailwind.config.ts

### Forbidden
- packages/core/**
- packages/tui/**

## Requirements
1. Vite setup with React DOM and TypeScript
2. Tailwind CSS with mobile-first configuration
3. PWA manifest with icons and theme colors
4. Service worker for offline shell
5. Connection setup screen (server URL + token input)
6. Dark mode support with system preference detection

## Definition of Done
- [ ] Mobile app builds and runs
- [ ] PWA manifest generated
- [ ] Service worker caches offline shell
- [ ] Connection setup functional
- [ ] Dark mode toggles correctly
